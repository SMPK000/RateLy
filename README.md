# RateLy

Telegram bot for currencies, gold, crypto, charts, support, referral, and admin tools.

## Environment variables

- BOT_TOKEN
- BOT_USERNAME
- ADMIN_IDS
- BOT_NAME
- ADS_TEXT (optional)
- ADS_EVERY (optional, default 0)
- DEFAULT_LANG (optional, default en)

## Run locally

```bash
pip install -r requirements.txt
python main.py
```
