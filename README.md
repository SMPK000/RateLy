# RateLy

Telegram bot for live crypto / currency / metal prices and charts.

## Files
- main.py
- requirements.txt
- render.yaml

## Environment variables
- BOT_TOKEN
- BOT_USERNAME
- BOT_NAME
- ADMIN_IDS
- DEFAULT_LANG
- ADS_TEXT
- ADS_EVERY

## Deploy
1. Upload to GitHub.
2. Connect the repo to Render as a Web Service.
3. Set the environment variables.
4. Deploy.

## Group usage
- Turn off Telegram bot privacy mode in BotFather to read normal group messages.
- Add the bot to your group.

## Commands
- /start
- /price BTC
- /chart BTC 7d
- /support message
- /panel
- /stats
- /broadcast message
- /ban user_id
- /unban user_id
- /reset user_id
- /reply ticket_id message
- /id
