# RateLy

A multilingual Telegram bot for live prices, charts, support tickets, admin tools, and scheduled broadcasts.

## Environment variables
- BOT_TOKEN
- BOT_USERNAME
- BOT_NAME
- ADMIN_IDS
- DEFAULT_LANG
- ADS_TEXT
- ADS_EVERY
- APP_TITLE
- PYTHON_VERSION=3.11.9

## Render
Build Command: `pip install -r requirements.txt`

Start Command: `python main.py`

## BotFather
Turn off Privacy Mode if you want the bot to read normal group messages.

## Commands
/start, /help, /lang, /base, /price, /chart, /support, /reply, /admin, /stats, /user, /ban, /unban, /vip, /reset, /broadcast, /broadcastin, /tickets
